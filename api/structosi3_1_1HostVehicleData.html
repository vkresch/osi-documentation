struct HostVehicleData¶

Interface for host vehicle data that is available to sensors and other functions due to host vehicle’s internal communication.

All coordinates and orientations are relative to the global ground truth coordinate system.

Public Members

optional BaseMoving osi3::HostVehicleData::location = 1

Current estimated location based on GPS- and related navigation sensors.

Note

Note that dimension and base_polygon need not be set.

optional BaseMoving osi3::HostVehicleData::location_rmse = 2

Current estimated location error based on GPS- and related navigation sensors.

Note
